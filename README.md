# 📝 React To-Do List App

This is a modern and stylish To-Do List application built using React + Vite. It allows users to:

-  Add tasks
-  Mark tasks as complete
-  Delete tasks
-  Edit tasks
-  Filter tasks (All / Active / Completed)
